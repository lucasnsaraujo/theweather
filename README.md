# THE NEWS
A news website made with React (still in development)

---
## How to start
1. `npm install` to install all dependencies 
2. `npm start` to start the project (it usually runs on port 3000)
---
## Technologies used
- React + Hooks
- Context API
- dotenv (for development variables)
